ReGenESyS - REborned GENeric and Expansible SYstem Simulator  v22.01 (wealljudes)
Genesys is a result of teaching and research activities of Professor Dr. Ing Rafael Luiz Cancian. It began in early 2002 as a way to teach students the basics and simulation techniques of systems implemented by other comercial simulation tools, such as Arena. In Genesys development he replicated all the SIMAN language, used by Arena software, and Genesys has become a clone of that tool, including its graphical interface. Genesys allowed the inclusion of new simulation components through dynamic link libraries and also the parallel execution of simulation models in a distributed environment. The development of Genesys continued until 2007, when the professor stopped teaching systems simulation classes. Ten years later the professor starts again to teach systems simulation classes and to carry out scientific research in the area. So in 2018 Genesys is reborn as ReGenesys, with new language and programming techniques, and even more ambitious goals.

203 void Seize::_execute(Entity* entity) {
204  for (SeizableItem* seizable : *_seizeRequests->list()) {
205  Resource* resource = _getResourceFromSeizableItem(seizable, entity);
206  unsigned int quantity = _parentModel->parseExpression(seizable->getQuantityExpression());
207  if (resource->getCapacity() - resource->getNumberBusy() < quantity) { // not enought free quantity to allocate. Entity goes to the queue
208  WaitingResource* waitingRec = new WaitingResource(entity, _parentModel->getSimulation()->getSimulatedTime(), quantity, this);
209  Queue* queue;
210  if (_queueableItem->getQueueableType() == QueueableItem::QueueableType::QUEUE) {
211  queue = _queueableItem->getQueue();
212  } else { // assume SET
213  Set* set = _queueableItem->getSet();
214  unsigned int index = _parentModel->parseExpression(_queueableItem->getIndex());
215  _parentModel->getTracer()->traceSimulation(this, "Member of set " + set->getName() + " chosen index " + std::to_string(index), Util::TraceLevel::L8_detailed);
216  queue = static_cast<Queue*> (set->getElementSet()->getAtRank(index));
217  }
218  queue->insertElement(waitingRec); // ->list()->insert(waitingRec);
219  _parentModel->getTracer()->traceSimulation(this, _parentModel->getSimulation()->getSimulatedTime(), entity, this, "Entity starts to wait for resource in queue \"" + queue->getName() + "\" with " + std::to_string(queue->size()) + " elements");
220  return;
221  } else { // alocate the resource
222  resource->seize(quantity, _parentModel->getSimulation()->getSimulatedTime());
223  _parentModel->getTracer()->traceSimulation(this, _parentModel->getSimulation()->getSimulatedTime(), entity, this, entity->getName() + " seizes " + std::to_string(quantity) + " elements of resource \"" + resource->getName() + "\" (capacity:" + std::to_string(resource->getCapacity()) + ", numberbusy:" + std::to_string(resource->getNumberBusy()) + ")");
224  }
225  }
226  _parentModel->sendEntityToComponent(entity, this->getConnections()->getFrontConnection());
227 }
